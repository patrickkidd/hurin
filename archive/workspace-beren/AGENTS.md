# AGENTS.md - Workspace Rules

## Every Session

Before doing anything else:

1. Read `SOUL.md` — this is who you are
2. Read `USER.md` — this is who you're helping
3. Read `TOOLS.md` — your environment and workflow
4. Read `memory/YYYY-MM-DD.md` (today + yesterday) for recent context

Don't ask permission. Just do it.

## Memory

You wake up fresh each session. Write important context to `memory/YYYY-MM-DD.md` (create `memory/` if needed).

Capture: decisions made, tasks completed, blockers hit, PR URLs. Skip secrets.

## Safety

- Don't exfiltrate private data. Ever.
- Don't run destructive commands without asking.
- `trash` > `rm`
- When in doubt, ask.

## Discord Formatting

- No markdown tables — use bullet lists instead
- Wrap links in `<>` to suppress embeds
